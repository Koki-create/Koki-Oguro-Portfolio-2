私の特長やスキル

～ 特長 ～

1. 貪欲な学習意欲

業務時間中だけでなく、業務時間外でも毎日最低3時間の学習時間を確保して日々積極的に学び、得た知識を業務に活かしております。現在の担当業務では、自発的に学んだExcelの知識を活用し、時には先輩社員の力を借りつつ、担当業務に関数やマクロを導入しました。その結果、月に100時間ほど要していた作業時間を20時間にまで短縮し、さらには作業ミスを撲滅できました。

2. 根本から解決する姿勢

物事について、仕組み・環境から見直して再構築し、問題を根本から解決していくことが得意です。例えば、担当していたお問い合せ対応業務では、"そもそもなぜお問い合わせが発生するのか？"と根本から考え、分析し、仕組みを見直すことで、月5件以上発生していたお問い合わせの数を月0〜1件程度まで削減し、業務対応時間の削減に成功しました。

3. 英語力

大学時代、1年間の交換留学をしていた際に、英語でのコミュニケーション力を身に付けました。最初は、周りの人たちの英語の会話に全くついていけない自分に絶望し、悔しさから文字通り涙することもありました。しかし、その悔しさを糧に、24時間英語漬けの環境をつくることで、猛スピードで英語力を磨き、ネイティブスピーカーとも会話できるレベルとなりました。

Koki Oguro's Portfolio



今ご覧頂いている、このWebサイトです。ネットで見つけたテンプレートをベースに使用しており、文字や画像のアニメーション、アイコンなど細部は自分でコードを書いて実装しました。
彩りを加える、より動きをつけるなど、取り入れたいアイディアがまだあるので、追加していき、サイトを訪れた方が自然と滞在して見入ってしまうサイトに仕上げていきたいと考えております。

使用言語など：HTML/CSS、Bootstrap、JavaScript

Djangoブログアプリ



フレームワークにDjangoを使用している、ブログのWebアプリです。簡単なアプリですが、CRUD機能・ページネーション機能・画像ファイルのアップロード機能など、Webアプリで使われることが多い機能がひと通り組み込まれております。今後は、テストコードの追加やAWSの導入などによって、さらに機能を充実させていきつつ、これらの技術を実務に活かしていければと思います。

使用言語など：HTML/CSS、Sqlite、Django、Python 3.11.4、Bootstrap
GitHub

画面の位置で変わる動物



マウスの位置に動物の顔が表示され、マウスを動かすと、その位置に応じて、表示される動物の顔が変わるアプリです。Processingのおかげで、少ない行数の簡単なコードを書くだけで、絵が描けてしまうので、プログラミングをより身近に、そしてその可能性を感じさせてくれるアプリです。

使用言語など：Python 3.11.4、Processing 4.2
GitHub

Youtube分析ツール



何かしらのキーワードを入力して検索し、表示されたYoutube上の動画を、さらにチャンネル登録者数でフィルタをかけて表示できる、Youtubeの分析ツールです。YouTube Data APIを用いており、フレームワークにはStreamlitを使用しております。さらに機能を充実させていき、Youtubeチャンネルを運営する人が、より高度な分析を行えるツールへと進化させていければと考えております。

使用言語など：Streamlit 1.23.1、Python 3.11.4
GitHub
